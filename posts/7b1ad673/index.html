еүҚиЁҖ

дј з»ҹж–№ејҸпјҲйә»зғҰпјүпјҡ

  • еҶҷе®Ңж–Үз«  вҶ’ жң¬ең°жү§иЎҢ hexo clean && hexo g && hexo d вҶ’ зӯүеҫ…еҮ еҲҶй’ҹз”ҡиҮіжӣҙд№… вҶ’ еҚҡе®ўдёҠзәҝ
  • йҡҸзқҖж–Үз« еўһеӨҡпјҢжҜҸж¬ЎйғЁзҪІж—¶й—ҙи¶ҠжқҘи¶Ҡй•ҝпјҢеҸӘиғҪе№ІзӯүзқҖ

иҮӘеҠЁеҢ–ж–№ејҸпјҲиҪ»жқҫпјүпјҡ

  • дҪ еҶҷе®Ңж–Үз«  вҶ’ git add . && git commit -m "ж–°ж–Үз« " && git push вҶ’ з§’дј е®ҢжҲҗпјҢеҚҡе®ўиҮӘеҠЁдёҠзәҝ

жҠҠиҝҷдёӘиҝҮзЁӢжҜ”дҪңеӨ–еҚ–й…ҚйҖҒпјҡ

  1. дҪ зҡ„жң¬ең°з”өи„‘ = дҪ зҡ„е®¶пјҲеҶҷж–Үз« зҡ„ең°ж–№пјү
  2. жҷ®йҖҡз§Ғжңүд»“еә“ = еӨ–еҚ–е№іеҸ°пјҲеӯҳеӮЁдҪ зҡ„еҺҹе§ӢеҶ…е®№пјү
  3. Page Repository = йӨҗеҺ…пјҲз”ЁжҲ·и®ҝй—®зҡ„зҪ‘з«ҷпјү
  4. GitHub Actions = й…ҚйҖҒе‘ҳпјҲиҮӘеҠЁеҢ–е·Ҙе…·пјү

ж•ҲзҺҮжҸҗеҚҮпјҡ

  • жң¬ең°еҸӘйңҖиҰҒ git ж“ҚдҪңпјҲеҮ з§’й’ҹпјү
  • жңҚеҠЎеҷЁдёҠзҡ„ GitHub Actions жңүжӣҙејәзҡ„и®Ўз®—иғҪеҠӣ
  • дҪ еҸҜд»Ҙз«ӢеҚіеҺ»еҒҡе…¶д»–дәӢпјҢдёҚз”Ёзӯүеҫ…йғЁзҪІе®ҢжҲҗ

еўһйҮҸдјҳеҢ–пјҡ

  • GitHub Actions дјҡжҷәиғҪең°еҸӘеӨ„зҗҶеҸҳжӣҙзҡ„йғЁеҲҶ
  • йҒҝе…ҚдәҶжҜҸж¬ЎйғҪйҮҚж–°з”ҹжҲҗжүҖжңүж–Үд»¶зҡ„жөӘиҙ№

з®ҖеҚ•иҜҙпјҢе°ұжҳҜжҠҠеҺҹжқҘйңҖиҰҒдҪ жүӢеҠЁеҒҡзҡ„йҮҚеӨҚеҠіеҠЁпјҢдәӨз»ҷдәҶ GitHub зҡ„жңәеҷЁдәәиҮӘеҠЁе®ҢжҲҗпјҢиҖҢдҪ еҸӘйңҖиҰҒдё“жіЁдәҺеҶҷдҪңжң¬иә«гҖӮ

иҙҰеҸ·дёҺз§Ғжңүд»“еә“

иҺ·еҸ–иҙҰеҸ·tokenпјҢз§Ғй’Ҙе’Ңе…ұй’ҘпјҢе·Із»Ҹжңүзҡ„еҸҜд»Ҙи·іиҝҮжӯӨжӯҘ

й…ҚзҪ®ж•ҷзЁӢ

з§Ғй’Ҙпјҡid_rsaпјҢе°Ҷз§Ғй’Ҙй…ҚеҲ°д»“еә“еҸҳйҮҸпјҢдёӢйқўдјҡи®ІеҲ°

е…¬й’Ҙпјҡid_rsa.pubпјҢе°Ҷе…¬й’Ҙй…ҚзҪ®еҲ°githubиҙҰеҸ·пјҢдёҠиҝ°й“ҫжҺҘжңүиҜҙ

  1. зҷ»еҪ•githubиҙҰеҸ·пјҢж–°е»әд»“еә“
  2. д»“еә“еҗҚпјҢиҮӘе·ұйҡҸдҫҝеҶҷпјҢжҲ‘иҫ“зҡ„zdbs
