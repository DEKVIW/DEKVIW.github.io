GithubйЎ№зӣ®ең°еқҖ

  1. ж–°еўһиҜҰз»ҶйЎөпјҡ
    дҪҝз”ЁжңҚеҠЎеҷЁеңЁзәҝзҲ¬иҷ«жҠҖжңҜгҖҒз¬¬дёүж–№ API и°ғз”ЁгҖҒеҗҺз«Ҝ API жҗңзҙўиҺ·еҸ–зӯүдёүз§ҚдёҚеҗҢзҡ„еӣһйҖҖжңәеҲ¶пјҢзЎ®дҝқж•°жҚ®иҺ·еҸ–зҡ„зЁіе®ҡжҖ§гҖӮ
  2. ж”ҜжҢҒжөҸи§Ҳж»ҡеҠЁи®°еҪ•пјҡ
    иҝӣе…ҘиҜҰз»ҶйЎөжҹҘзңӢеҪұзүҮеҗҺпјҢеҰӮжһңдёҚе–ңж¬ўйҖҖеҮәдёҠзә§йЎөйқўпјҢеҸҜд»ҘеӣһеҲ°д№ӢеүҚзҡ„ж»ҡеҠЁдҪҚзҪ®пјҢдёҚдјҡдёўеӨұжөҸи§Ҳи®°еҪ•гҖӮ
  3. е…ЁйқўйҮҚжһ„дәӨдә’йҖ»иҫ‘гҖҒеҗҺз«Ҝд»Јз ҒеҸҠ UIпјҡ
    еҗҢж—¶дјҳеҢ–жЎҢйқўз«ҜдёҺз§»еҠЁз«Ҝзҡ„дәӨдә’йҖ»иҫ‘пјҢдҪҝе…¶жӣҙдәәжҖ§еҢ–гҖҒжӣҙз¬ҰеҗҲз”ЁжҲ·д№ жғҜпјҢжҳҫи‘—жҸҗеҚҮдҪҝз”ЁдҪ“йӘҢгҖӮ
    ж’ӯж”ҫйЎөйқўзҡ„йӣҶж•°еҸҜйҖҡиҝҮжҢүй’®жҲ–жӮ¬еҒңжҹҘзңӢе®Ңж•ҙдҝЎжҒҜпјҲйҒҝе…Қз»јиүәй•ҝж ҮйўҳеҪұе“ҚйҖүйӣҶеҠҹиғҪпјүгҖӮ
    з«ҷзӮ№ API й…ҚзҪ®йЎөж”ҜжҢҒ API жңүж•ҲжҖ§жөӢиҜ•дёҺиҮӘеҠЁжҺ’еәҸеҠҹиғҪгҖӮ
  4. йҮҚжһ„и§ӮзңӢи®°еҪ•йҖ»иҫ‘пјҢдјҳеҢ–и®°еҪ•ж–№ејҸдёҺи°ғз”ЁйҖ»иҫ‘гҖӮ
  5. йҮҚжһ„ж”¶и—ҸйЎөйқўпјҢжҸҗеҚҮдҪҝз”ЁдҪ“йӘҢдёҺе“Қеә”йҖҹеәҰгҖӮ
  6. дјҳеҢ–жҗңзҙўйЎөйқўпјҡ
    • е®һзҺ°зғӯй—ЁжҺЁиҚҗеҠҹиғҪпјҲз»“еҗҲж•°жҚ®еә“зғӯжӣҙж–°жңәеҲ¶пјҢе®һж—¶жҺЁиҚҗеҶ…е®№пјүгҖӮ
    • е®һзҺ°жөҒејҸжҗңзҙўпјҢжҸҗй«ҳе“Қеә”йҖҹеәҰдёҺжҗңзҙўдҪ“йӘҢгҖӮ
    • йҮҚжһ„жҗңзҙўз»“жһңжҺ’еәҸйҖ»иҫ‘дёҺиҒҡеҗҲйҖ»иҫ‘гҖӮ
  7. дјҳеҢ–еҠ иҪҪйЎөйқўпјҡ
    жҸҗеҚҮеҠ иҪҪйҖҹеәҰдёҺи§Ҷи§үи¶Је‘іжҖ§гҖӮ
  8. еўһејәдјҳйҖүи·ҜзәҝиҜ„еҲҶжңәеҲ¶пјҡ
    е®Ңе…ЁйҮҚжһ„йҖ»иҫ‘пјҢзЎ®дҝқдјҳйҖүи·Ҝзәҝзҡ„жңүж•ҲжҖ§дёҺз§‘еӯҰжҖ§пјҢжҸҗй«ҳж’ӯж”ҫдјҳйҖүйҖҹеәҰдёҺзЁіе®ҡжҖ§гҖӮ
  9. дҝ®еӨҚеҶ…еӯҳжі„жјҸй—®йўҳпјҢжҳҫи‘—жҸҗй«ҳзі»з»ҹзЁіе®ҡжҖ§гҖӮ
  10. з§»йҷӨеҶ—дҪҷеҠҹиғҪпјҡ
    зІҫз®Җж— з”ЁжЁЎеқ—пјҢеҮҸе°‘еҶ—дҪҷеәҰпјҢжҸҗй«ҳзҪ‘з«ҷе“Қеә”йҖҹеәҰпјҢдҪҝеёғеұҖжӣҙеҠ еҗҲзҗҶз®ҖжҙҒгҖӮ
  11. еӨҡз«ҜдјҳеҢ–пјҡ
    й’ҲеҜ№з§»еҠЁз«ҜдёҺжЎҢйқўз«ҜеҲҶеҲ«дјҳеҢ–пјҢз¬ҰеҗҲдёҚеҗҢи®ҫеӨҮзҡ„жңҖдҪідәӨдә’йҖ»иҫ‘гҖӮ
  12. йҮҚжһ„ UI дёҺеёғеұҖпјҢеҠ е…ҘеҠЁжҖҒж•ҲжһңпјҢж•ҙдҪ“йЈҺж јжӣҙзҺ°д»ЈгҖҒжӣҙжөҒз•…гҖӮ
  13. зҷ»йҷҶйЎөйқўдјҳеҢ–пјҡ
    ж–°еўһвҖңи®°дҪҸжҲ‘вҖқдёҺвҖңжҹҘзңӢеҜҶз ҒвҖқеҠҹиғҪгҖӮ
  14. еўһејәе№ҝе‘ҠиҝҮж»ӨйҖ»иҫ‘пјҡ
    еӨ§е№…жҸҗеҚҮе№ҝе‘ҠжӢҰжҲӘжңүж•ҲжҖ§пјҢе®һзҺ°жӣҙеҪ»еә•зҡ„е№ҝе‘ҠеұҸи”ҪгҖӮ
  15. ж–°еўһеҪұзүҮеҶ…е®№йў„еҠ иҪҪеҠҹиғҪпјҡ
    жҸҗй«ҳеҠ иҪҪж•ҲзҺҮпјҢе®һзҺ°ж— ж„ҹеҠ иҪҪпјҢеҮҸе°‘е»¶иҝҹж„ҹпјҢеҗҢж—¶йҷҚдҪҺ API и°ғз”Ёйў‘зҺҮпјҢеҮҸиҪ»жңҚеҠЎеҷЁеҺӢеҠӣгҖӮ
  16. жӣҙеӨҡдјҳеҢ–дёҺдҝ®еӨҚпјҡ
    еҮ д№ҺеҜ№еҺҹжңүйЎ№зӣ®иҝӣиЎҢдәҶвҖңжүӢжңҜзә§вҖқйҮҚжһ„пјҢж–°еўһеӨҡйЎ№еҠҹиғҪгҖҒдҝ®еӨҚиӢҘе№І BUGгҖӮ
