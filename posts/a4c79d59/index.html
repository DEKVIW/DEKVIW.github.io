一览成文桌面版：从视频到文章不必再碰命令行

发表2026-07-16|更新2026-07-17|软件
|字数总计:1.5k|阅读时长:4分钟

前言

上一篇 《从视频到文章：AI 自动生成美食教程》 ，命令行用起来还是不太方便

所以有了桌面版：

一览成文（YilanChengWen）
绿色包解压 → 双击 YilanChengWen.exe → 图形界面完成「视频 → 文章」。

示例成稿（美食提示词产出）可参考：一览美食

下载与开源

入口 链接 说明
GitHub 项目 DEKVIW/video-to-article 源码、README、Issue
GitHub Release Releases 主程序绿色包 zip（含 FFmpeg，不含语音大模型权重）
夸克网盘（弱网推荐） pan.quark.cn 分享 软件本体 + 转写模型整包，网络不稳时更方便

建议程序解压到纯英文路径（如 D:\Apps\YilanChengWen\）。本地转写需要 FunASR 模型时，将模型包中的 models 合并到 exe 同级，确认存在：

路径含中文时，不要默认「只拷到程序同级就一定能读」——更稳妥是英文路径，或把模型放到同盘 盘符:\YilanChengWenData\models\funasr\。详见项目 README。

能做什么

输入来源

类型 说明
在线链接 能力随 yt-dlp 更新，常见包括 B 站、YouTube、抖音、小红书、微博 等
本地文件 本机音频 / 视频直接转写
批量 链接清单、本地目录、部分合集场景
仅下载 只要媒资 / 字幕，可跳过转写与成稿
视频搜索 如 B 站搜索到结果后再勾选处理
补跑工具 已有 raw 时补文章、重生成封面、检查 / 刷新 cookies 等

处理与产出

  • 字幕优先：部分平台可拉字幕，失败再回退 本地 ASR
  • 默认 ASR：FunASR / SenseVoice（偏中文口播）；可选 Whisper
  • 大模型成稿：按提示词模板生成结构化文章（单条视频即可自选 / 自定义模板，见下文）
  • 可选 AI 封面：仅导出提示词 / 调用生图 API（在设置中配置）
  • 结果目录：output\ 下 Markdown 等；界面可「打开结果」

边界

  • 不是实时直播同传字幕产品
  • 需要自备 大模型 API Key（仅下载、仅转写可不配）
  • YouTube 等平台常需 cookies / 登录态，风控变化时需自行更新

处理流程

软件界面对应关系：

  1. 单条处理 — 一条链接或本地文件成文，并可当场选择提示词模板
  2. 批量 — 清单 / 本地目录
  3. 仅下载 — 只要文件不要文章
  4. 文章模板 — 工作台下拉选择；文件在 prompts\articles\，可自建
  5. 设置 — API、转写默认、cookies、封面等

自定义提示词

成稿长什么样，不由程序写死，而由你选的提示词模板决定。
默认自带偏美食教程的模板（如 snack_recipe），也可以换成自己的模板。同一段转写，换不同模板，会得到不同结构与语气（食谱、提纲、访谈整理、评测稿等）。

新增模板

  1. 进入程序目录：prompts/articles/
  2. 新建 .md 文件（文件名会作为下拉选项）

快速上手

解压即用

解压绿色包 zip 到目标目录，例如：

不需要安装到系统；删掉整个文件夹即卸载。

语音模型

  • 方式 A（弱网）：夸克 / 模型 zip，把 models 合并到 exe 同级（不要多套一层目录）
  • 方式 B：首次转写时自动从 ModelScope 下载（需能访问）
  • 仅下载视频/字幕、且平台字幕可用时，可不装模型

大模型 API

打开程序 设置 → 大模型：

  • 兼容 OpenAI 形态的 Chat 接口：API Key / Base URL / 模型名
  • 密钥写在程序旁 config.json，不要把含 Key 的配置打包公开分享

跑通一条

  1. 单条处理 → 粘贴链接或选本地文件
  2. 选文章模板（内置或自建均可；单条就能自定义，不必先批量）
  3. 点 开始，看运行日志
  4. 到 output\ 取 Markdown；需要时点「打开结果」

建议先用 1 条 试跑（并可换一个自建模板），再批量。

  1. 单条处理 — 一条链接或本地文件成文，并可当场选择提示词模板
  2. 选文章模板（如美食教程）
  3. 点 开始，看运行日志
  4. 到 output\ 取 Markdown；需要时点「打开结果」

运行环境提示

  • FFmpeg：绿色包优先用程序目录 ffmpeg\；缺失时再装系统版并加入 PATH
  • 磁盘：程序 + 模型 + 缓存建议预留数 GB
  • 内存：本地转写时占用会明显高于纯下载

主要配置

位置 配什么
设置 → 大模型 Provider / API Key / Base URL / 模型名 / 超时与重试
设置 → 转写 默认 ASR（FunASR / Whisper）、FunASR 模型名、线程、FunASR 模型目录（可留空自动）
设置 → YouTube / 下载 默认 cookies 浏览器或 cookies 文件、PO Token（按需）
设置 → AI 封面 是否启用生图、仅导出提示词、API 与尺寸等（可选）
工作台 · 文章模板 对应 prompts\articles\；单条/批量都可选，决定成稿结构
工作台 · 封面策略 跟随设置 / 本次启用封面 / 仅导出提示词 / 全不勾=本次关闭
工作台 · Cookies 单次任务覆盖设置中的默认 cookies
工作台 · 高级 ASR 仅本次覆盖引擎参数
